# 전일민평 대시보드

`전일민평종가표.csv`를 기반으로 **스팟 금리 커브**와 **기대수익률(ER) 커브**를 키워드로 필터링해 시각화하는 로컬 대시보드입니다.

## 실행 방법

1) 패키지 설치

```bash
python -m pip install -r requirements.txt
```

2) 대시보드 실행

```bash
python -m streamlit run dashboard.py
```

## 기능

- 사이드바에서 지표 선택: **스팟 금리 / 기대수익률**
- `종목명` 키워드 포함/제외 필터
- `채권그룹` 텍스트 포함 필터 (예: `AAA`)
- 최대 만기 선택, y축 상단 지정, 최대 라인 수 제한

## 기대수익률 정의

기대수익률(ER)은 아래 정의로 계산합니다.

\[
ER(t)=y(t) + (y(t)-y(t-1y)) \times Duration(t)
\]

- CSV에 듀레이션 컬럼이 없어서, 기본은 **Duration(t)=만기(년)** 으로 근사합니다.
- `t-1y`는 파일에 존재하는 만기들 중 “1년 앞”에 해당하도록 매핑합니다.
